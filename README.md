
# تطبيق تنظيم الإجازات

تطبيق ذكي لإدارة الوقت والإجازات مع دعم كامل للغة العربية والإنجليزية.

## كيفية تشغيل البرنامج

### الطريقة الأولى: تشغيل محلي (مطلوب Node.js)

1. تأكد من تثبيت Node.js على جهازك
2. افتح Terminal أو Command Prompt في مجلد المشروع
3. قم بتثبيت المكتبات:
```bash
npm install
```

4. شغل البرنامج:
```bash
npm run dev
```

5. افتح المتصفح على العنوان: http://localhost:3000

### الطريقة الثانية: بناء للإنتاج

```bash
npm run build
npm run preview
```

## ملاحظة مهمة

⚠️ **لا يمكن فتح ملف index.html مباشرة في المتصفح**

هذا التطبيق مبني بتقنية React + Vite ويحتاج إلى خادم محلي للعمل بشكل صحيح. 

استخدم الأوامر المذكورة أعلاه لتشغيل البرنامج.
